Solucionador multi-threads para laberintos

    En este proyecto se desarrolló un programa en C para Linux que simula el recorrido de un laberinto utilizando hilos (threads).
    El objetivo principal del programa es explorar un laberinto, utilizando los hilos para explorar todos los posibles caminos y verificar si existe una, ninguna o varias salidas.

Características del proyecto

  • El código utiliza una estructura de datos para representar un laberinto, con una matriz de celdas donde cada carácter representa una pared, espacio vacío o la meta.
  • Implementa un sistema de hilos utilizando la biblioteca pthread, donde cada hilo representa la exploración de un posible camino en el laberinto.
  • Los hilos se mueven en el laberinto de manera autónoma, avanzando en una dirección específica hasta encontrar una pared o la meta.
  • Utiliza mutexes para sincronizar el acceso a los datos compartidos entre hilos.
  • Muestra en la consola el laberinto con las posiciones de los hilos en tiempo real.
